# Hooks Library — مكتبة الـ Hooks
**المرجع الرئيسي لـ Karim — قبل ما تكتب hook راجع الأنواع دي**

---

## إزاي تستخدم الملف ده

```
1. حدد الـ brand + الـ platform
2. اختار النوع المناسب للـ angle اللي عندك
3. طبّق الـ pattern على الفكرة — مش بتنسخ، بتستخدم الـ structure
```

---

## النوع الأول — PAIN HOOKS (الأكثر فاعلية للـ SMBs)

```
"[مشكلة يعيشها يومياً] — وده مش طبيعي"
"[عدد] ساعة في [مهمة عادية] كل أسبوع — إمتى هيخلص؟"
"كل الـ [role] بيعانوا من [مشكلة] — غير واحد"
"[نتيجة سيئة] مش لأنك غلطان — لأن [السبب الحقيقي]"
"[مشكلة] بتكلفك [تكلفة] وأنت مش واخد بالك"
```

**يشتغل مع:** Value Plus / ZOZ / AAMBFS
**مش مناسب لـ:** أثر وحكمة / مكث

---

## النوع الثاني — RESULT HOOKS (للـ Personal Brand)

```
"[نتيجة مباشرة] — وده حصل في [وقت قصير]"
"[رقم محدد] في [وقت محدد] بـ [طريقة غير متوقعة]"
"جربت [حاجة] — الـ [مقياس] اتغير من [رقم] لـ [رقم]"
"الأسبوع اللي فات [نتيجة] — إيه اللي اتغير؟"
"[عدد] مرة جربت [حاجة] قبل ما أعرف [الـ insight]"
```

**يشتغل مع:** Personal Brand / Dotz
**على أي platform**

---

## النوع الثالث — CONTRAST HOOKS (للـ insight content)

```
"الكل بيقول [X] — اللي بيشتغل فعلاً هو [Y]"
"مش [السبب اللي بيظنه الناس] — ده [السبب الحقيقي]"
"[شيء يبدو صح] وده بالظبط المشكلة"
"[نصيحة شائعة] vs [اللي بيحصل فعلاً]"
"[نتيجة إيجابية] جاءت من [قرار يبدو غلط]"
```

**يشتغل مع:** Personal Brand / AAMBFS
**أقوى على:** LinkedIn / X

---

## النوع الرابع — STORY HOOKS (للـ storytelling content)

```
"[لحظة محددة] — مش [موقف عام]"
"في [مكان محدد]، [وقت محدد]، [شيء حصل]"
"كنت [في موقف] لما [حاجة غير متوقعة] حصلت"
"[شخص] قالي [جملة] — وفضلت فاكرها"
"أول ما [حاجة] حصلت، [ردة فعل بشرية] — وبعدين..."
```

**يشتغل مع:** أثر وحكمة / Personal Brand
**أقوى على:** Facebook / Instagram

---

## النوع الخامس — CURIOSITY HOOKS

```
"في [موضوع] حاجة مش الكل بيعرفها"
"الـ [رقم/نسبة] دي بتفاجئ الناس دايماً"
"إيه اللي [الناجحين في مجال] بيعملوه بشكل مختلف؟"
"[سؤال يخليك تكمل] — الإجابة مش اللي بتتوقعها"
"[وصف موقف غريب] — وده حصل فعلاً"
```

**يشتغل مع:** كل الـ brands
**أقوى على:** Facebook / Instagram

---

## النوع السادس — VALIDATION HOOKS (للـ FOMO)

```
"[عدد] ناس بيعملوا [حاجة] — إنت؟"
"لو [scenario يشبه حياة القارئ] — المقال ده ليك"
"[profile محدد] اللي بيواجه [تحدي] هيلاقي هنا [حل]"
"مناسب لـ [profile دقيق] — مش لكل الناس"
```

**يشتغل مع:** AAMBFS / Value Plus / Dotz
**أقوى على:** Facebook Ads / Landing Pages

---

## النوع السابع — DIRECT HOOKS (للـ Ads فقط)

```
"[مشكلة؟] — في حل"
"[نتيجة] في [وقت] — هنوريك إزاي"
"[brand/product] بيساعد [profile] على [نتيجة]"
"[سؤال مباشر] — إجابتك هتحدد باقي الـ content"
```

**يشتغل مع:** كل الـ clients
**خاص بـ:** Layla للـ Ads — مش للـ organic

---

## HOOK QUALITY CHECKLIST

```
✅ الـ hook = فكرة واحدة بس
✅ فيه شخص محدد أو لحظة محددة؟
✅ الـ curiosity gap واضح؟ (بيخلي الناس تكمل؟)
✅ مناسب للـ brand voice؟ (مكث ≠ urgency)
✅ مش بيبدأ بـ "السلام عليكم" أو "النهارده هنتكلم"
✅ أقل من 15 كلمة لو ممكن

❌ مش generic — "محتوى مهم جداً"
❌ مش clickbait — وعد بحاجة مش موجودة في المحتوى
❌ مش corporate — "يسعدنا أن نعلنكم"
```

```
10/10 → فيه emotion + specificity + curiosity gap
8-9   → فيه اتنين منهم
6-7   → فيه واحد بس — ارجع وحسّن
< 6   → ابدأ من نوع تاني
```
